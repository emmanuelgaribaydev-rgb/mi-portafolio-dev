Acerca de Mí

Portafolio de Proyectos

1. Red Social para Lectores y Escritores Caliope

Contexto: Proyecto Integrador colaborativo desarrollado bajo la metodología SCRUM en Generation México.

Propósito y Problema: Crear una plataforma digital que centralice la interacción entre escritores independientes y lectores, resolviendo la falta de espacios dedicados a la retroalimentación literaria.

  • Desafío: Gestionar de forma eficiente relaciones complejas de datos (usuarios, seguidores, me gusta, comentarios) sin perder rendimiento en las consultas.
  • Solución: Diseñé e implementé un modelo entidad-relación robusto con mapeos muchos a muchos (Many-to-Many) en la persistencia de datos y optimicé las consultas SQL.
  • Tecnologías: Java, Spring Boot, MySQL, Git, Metodologías Ágiles.
  • Ver Código en GitHub

2. MVP: Sistema de Gestión de Microcréditos

Contexto: Proyecto individual enfocado en lógica de negocio para el sector bancario y Fintech.

Propósito y Problema: Automatizar el proceso de simulación y almacenamiento de créditos financieros, eliminando errores de cálculo manuales y mejorando la transparencia de tasas.

  • Desafío: Garantizar la precisión en cálculos matemáticos de interés compuesto y generar tablas de amortización consistentes en el Backend.
  • Solución: Desarrollé algoritmos financieros puros en Java utilizando tipos de datos de alta precisión (BigDecimal) y expuse los resultados mediante una API REST.
  • Tecnologías: Java, Spring Boot, API REST, Persistencia de Datos, Railway/Render.
  • Ver Código en GitHub

3. Sistema Integrado de Gestión de Librería

Contexto: Proyecto práctico de consolidación para el manejo de estructuras de datos y persistencia.

Propósito y Problema: Resolver los problemas de pérdida de inventario en un entorno comercial mediante un sistema CRUD estricto.

  • Desafío: Prevenir inconsistencias lógicas, como permitir el préstamo de un libro cuando el stock disponible se encuentra en cero.
  • Solución: Incorporé capas de validación lógica condicional en los servicios de Java previos a la ejecución de sentencias SQL (CRUD avanzado), asegurando la integridad del inventario.
  • Tecnologías: Java SE, JDBC, SQL, Consultas Estructuradas.
  • Ver Código en GitHub

4.MVP Módulo Algorítmico de Enmascaramiento (Cifrado César)

Contexto: Módulo enfocado en lógica algorítmica pura y fundamentos de criptografía básica aplicada.

Propósito y Problema: Mitigar los riesgos de filtración de datos sensibles en tránsito dentro de sistemas de software simulando entornos de seguridad bancaria.

  • Desafío: Diseñar una lógica de enmascaramiento cíclica eficiente basada en el Cifrado César pensado a escalar a cifrado por RSA sin depender de librerías externas de seguridad.
  • Solución: Desarrollé una solución algorítmica optimizada utilizando manipulación de arreglos y operaciones aritméticas de residuos (módulo) para gestionar el desfase de caracteres.
  • Tecnologías: Java Estándar (Lógica pura), Algoritmia, Control de Calidad de Código.
  • Ver Código en GitHub
